Morse Code Translator (Java)
📖 Overview
This program provides a simple Morse code translator written in Java. It allows users to:

Convert text (letters and digits) into Morse code.

Convert Morse code back into readable text.

The program runs in a loop, giving users the option to continue translating until they choose to exit.

⚙️ Features
Text to Morse: Converts letters (A–Z), digits (0–9), and spaces into Morse code.

Morse to Text: Decodes Morse sequences back into readable characters.

Interactive Menu: Users select translation direction via a simple console menu.

Error Handling: Detects invalid Morse input and prompts the user accordingly.

▶️ Usage
Run the program.

Choose:

1 → Translate Morse code to text.

2 → Translate text to Morse code.

Enter your input.

Decide whether to exit or continue.
